1. Introduction to Java

Welcome to your journey into Java! This guide provides a beginner-friendly overview of Java fundamentals.

What is Java?

Java is a simple, modern, general-purpose, object-oriented programming language used to develop console, window, web, and mobile applications.

Key Features
  • Platform Independent: Write once, run anywhere (WORA) via JVM.
  • Supports Multiple Apps: Console, window, web, mobile.
  • Exception Handling: Manages runtime errors.
  • Multithreading: Concurrent task execution.
  • Database Programming: Supports connectivity.
  • Security: Built-in features.
  • Strictly Typed: Explicit data types.
  • Case Sensitive: Upper/lowercase matters.
Java’s Evolution

Developed in 1992 as "OAK" by James Gosling at Sun Microsystems. Renamed to Java in 1995; now maintained by Oracle.

2. Java Basics

Software Classification
  • System Software: e.g., OS – interfaces user and hardware.
  • Application Software: Depends on system software; includes:
    • Packages: Task-specific tools (e.g., MS Office).
    • Languages: Communication medium (e.g., Java).
Language Types
  • Low-Level (LLL): Hard to write/understand (e.g., machine code).
  • High-Level (HLL): Easy to write/understand (e.g., Java).
  • Middle-Level (MLL): Combines LLL and HLL (e.g., C).
Key Terms
  • Platform: Hardware + OS combination.
  • Framework: Pre-built classes/interfaces for development.

3. Setting Up Java

Java Directory Structure
  • JDK: Java Development Kit – tools for coding.
  • JRE: Java Runtime Environment – runs apps.
Environment Setup

Set PATH:

  1. Define JAVA_HOME (e.g., C:\Program Files\Java\jdk-21).
  2. Add %JAVA_HOME%\bin to system PATH.
  3. Verify: java -version.

4. Java Program Structure

Basic Structure 
class ClassName {
    public static void main(String[] args) {
        // Statements
    }
}
  • public: Access modifier.
  • static: No object needed.
  • void: No return value.
  • main: Entry point.
  • String[] args: Command-line arguments.
Compilation and Execution
  • Compile: javac FileName.java.
  • Run: java ClassName.
  • Note: Match file and class names; case-sensitive execution.
Output Statements
  • System.out.print("text"): Same line.
  • System.out.println("text"): New line.
  • System.out.printf("format", vars): Formatted.

7. Input in Java

Command-Line Arguments 
class Example {
    public static void main(String[] args) {
        int a = Integer.parseInt(args[0]);
        int b = Integer.parseInt(args[1]);
        System.out.println(a + b);
    }
}

Run: java Example 5 10

Using Scanner 
import java.util.Scanner;
class Example {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter a number: ");
        int num = sc.nextInt();
        System.out.println("You entered: " + num);
    }
}

Methods: nextInt(), nextFloat(), etc.

8. Sample Programs

Sum of Two Integers 
class Sum {
    public static void main(String[] args) {
        int a = 5, b = 10;
        int sum = a + b;
        System.out.println("Sum is: " + sum);
    }
}
Area of a Square 
import java.util.Scanner;
class Square {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter side: ");
        int side = sc.nextInt();
        int area = side * side;
        int perimeter = 4 * side;
        System.out.println("Area: " + area);
        System.out.println("Perimeter: " + perimeter);
    }
}
